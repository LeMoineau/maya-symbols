
# maya-symbols

npm module pour la génération et la lecture de l'écriture maya par Pierrot

## Lancement

Build le module si changements
```
npm run build
```

Lancer le module dans le terminal *node.js*
```
node
maya = require('./lib/index')
```

## Tests

```
npm run test
```

## Code quality

```
npm run lint
```

## Requirements

__common__
- uuid

__dev__
- typescript
- mocha
- tslint
